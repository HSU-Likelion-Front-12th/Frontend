# StoryBook 실행하는 방법

- 먼저 해당 github URL을 크론한다.
- StoryBook lib 설치 유무를 확인하고 상황에 따라 설치를 해준다.
```
npx init sb
```

StoryBook 실행 명령어

```
npm run storybook

```

스토리북 버전은 아래와 같다.

```
"@storybook/react": "^8.0.6"
```
