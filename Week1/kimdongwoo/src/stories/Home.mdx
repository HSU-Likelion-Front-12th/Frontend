# Home Component

이 컴포넌트는 기존 과제의 body에 해당 되는 컴포넌트이다. JSX에서는 body태그를 사용할 수 없다. 왜냐하면 JSX는 React 컴포넌트를 생성하는 것이고 body 태그는 HTML 문서의 최상위 레벨 요소이기 때문이다.
